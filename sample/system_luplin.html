루플리나 Luplina
 
'루플린A(또는 루플리나)'는 웹 애플리케이션 프레임워크입니다. 레이아웃 시스템, 스프링 기반의 백엔드 API 제공, 프런트엔드 API인 루플린O(또는 루플리노)를 제공하여 복잡하고 어려운 엔터프라이즈 서비스 개발에도 빠르고 안정적인 앱 개발이 가능합니다.
