리얼그리드 소개

리얼그리드 RealGrid
 
웹 환경에서 편집, 행 그룹, 필터, 정렬 등 다양하고 강력한 기능을 가진 새로운 데이터 그리드 컴포넌트* 입니다.
액티브액스(ActiveX)방식이 아닌 최신 웹 기술로 모든 브라우저에서 실행이 가능합니다.
마치 스프레드시트(Spreadsheet)를 다루듯이 개발이 쉽고, 편집이 자연스러워 직관적인 사용과 완성도 높은 기능을 제공합니다.
자세히

※ 데이터 그리드 컴포넌트
작은 의미에서는 셀이라는 그리드의 최소 단위, 큰 의미에서는 행과 열을 가지는 표에서 각종 수치와 계산식 함수 심지어 그래프나 차트, 값 예측, 자동화 기능까지 처리 가능하다.

주요기능

행과 컬럼 틀 고정

고정(Fixed) 옵션을 통해서 행 또는 컬럼을 고정할 수 있고 고정된 셀을 특별하게 다룰 수 있습니다. 고정된 셀을 편집이 불가능하게 하거나 스타일을 통해 다르게 표현할 수 있습니다.

복사 및 붙여넣기

리얼그리드는 선택된 하나 이상의 셀을 클립보드로 복사하거나, 하나 이상의 셀을 구성하는 클립보드 데이터를 그리드로 붙여 넣을 수 있습니다. 리얼그리드에서 Ctrl+C 나 Ctrl+V 키를 직접 입력해서 기능을 확인해 보십시오.

다양한 레이아웃

컬럼을 그룹으로 묶어 복잡한 컬럼 레이아웃을 표현할 수 있습니다. 사용자가 컬럼을 그룹 판넬에 끌어다 놓으면 ‘행 그룹핑(Row Grouping)’이 되고 Summary mode를 통해 그룹의 푸터에 표시할 수 있는 합계나 평균과 같은 값의 종류를 지정할 수 있습니다.

엑셀 변환

고정(Fixed) 옵션을 통해서 행 또는 컬럼을 고정할 수 있고 고정된 셀을 특별하게 다룰 수 있습니다. 고정된 셀을 편집이 불가능하게 하거나 스타일을 통해 다르게 표현할 수 있습니다.

직관적인 필터 기능

리얼그리드는 사용법이 쉽고 직관적인 필터 기능을 가지고 있습니다. 필터는 일치하는 값의 데이터만 보여주거나 조건 범위 안에 있는 데이터를 보여줄 때 유용합니다. 컬럼의 도메인 정보를 이용해 코드 추가 없이 엑셀의 필터 기능을 똑같이 구현할 수 있습니다.

대용량 데이터 처리

대용량 데이터를 웹에서 처리한다는 것은 단순히 로딩만의 문제는 아닙니다. 리얼그리드는 읽어온 데이터를 브라우저 상에서 빠르게 스크롤하거나 정렬, 그룹핑 할 수 있습니다.

웹 화면의 트랜잭션 업무지원

데이터 별로 가지고 있는 상태(Row State)를 이용하면 다른 화면으로 옮겨 다니지 않고 여러 행, 여러 셀에 대한 입력/수정/삭제를 일괄적으로 처리할 수 있습니다.

다양한 시각적 표현

아래의 이미지는 연도별 필드의 값을 Spark Line컬럼에 표시하는 예입니다. 실행시간에도 값이 변경되면 Spark line 컬럼에 즉시 반영됩니다.

컬럼 생성기

그리드 컬럼 구성과 필드 연결을 쉽게 할 수 있는 컬럼 생성기를 사용해 보세요. 컬럼 생성기는 엑셀과 구글의 스프레드 시트 애드온 프로그램으로 사용자에게 친숙한 엑셀에서 그리드의 레이아웃을 디자인하고 리얼그리드에서 사용할 수 있는 Javascript 소스를 만들어 줍니다. 엑셀용 컬럼 생성기가 필요하다면 기술지원에 요청해 주십시오.

피벗

신속한 데이터 분석은 비즈니스 결정을 내리는 데 도움이 됩니다. 하지만 데이터 양이 많거나 분석할 내용이 많으면 쉽게 알 수 없습니다. 리얼피벗은 데이터를 요약, 분석, 탐색 및 표시하는 훌륭한 방법을 제공합니다. 몇 번의 클릭으로 피벗 자료를 만들 수 있습니다. 리얼피벗은 매우 탄력적이어서 완성된 결과를 신속하고 다양하게 표시하는 방법 변경할 수 있습니다. 또한, 그리드를 기반으로 하여 그리드 데이타가 업데이트될 때 자동으로 피벗 결과를 수정해 보여줍니다.

원 데이터, 멀티 그리드

RealGrid의 독창적인 아키텍처는 하나의 데이터 셋(Provider)을 여러 그리드에서 공유해 사용할 수 있습니다. 그리드에 표시하는 컬럼의 순서나 렌더러를 다르게 할 수 있고, 필터나 그룹핑을 다르게 적용해 특별한 UI를 만들 수 있습니다. 데이터의 수정이 일어나면 같은 데이터 셋을 사용하는 각각의 그리드에 수정된 값이 즉시 적용됩니다.

컬럼 그룹

RealGrid는 컬럼 Grouping 말고도 대상 컬럼의 값을 기준으로 2가지 Grouping 방법을 제공합니다. 그 중 Merged Row Grouping은 묶여진 컬럼 셀들을 병합셀로 표시하고 그룹 마다 Header나 Footer를 추가하여 표시합니다. 특정 컬럼의 집계 값을 표현하는데 적합 합니다.

행 병합 그룹핑

RealGrid는 컬럼 Grouping을 통해 복잡한 Layout도 표현할 수 있습니다. 이 때 컬럼그룹에 속한 각 셀의 높이는 기본으로 균등하게 배분 되지만 속성 값을 수정하면 수직으로 배열되는 컬럼 그룹 내에서 각 컬럼의 높이를 다르게 할 수 있습니다. 그리드 컬럼 Layout은 파일이나 쿠키에 저장했다가 다음에 가져와 사용할 수 있어 Layout의 개인화가 가능합니다.

스파크 라인 렌더러

Series 컬럼에서 사용할 수 있는 Spark line renderer는 기간 등 일정 범위 내의 데이터 흐름을 간략하고 직관적으로 표시하는 데 사용합니다. 데이터 내용에 맞게 Line, Column, Win/Loss 타입의 Series 컬럼 Renderer를 선택할 수 있습니다.
